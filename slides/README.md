## Customizing

All visual constants are at the top of `make_deck.py`:
- `BG_DARK / BG_PANEL / TEXT / ORANGE` — palette
- `SLIDE_W / SLIDE_H / MARGIN` — geometry
- `FONT_FAMILY / MONO_FAMILY` — typefaces

Each slide is one function. To reorder or remove a slide, edit the `builders` list in `main()`. To add a slide, write a new `slide_*` function and append it.
